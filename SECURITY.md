# Security Policy

## Supported Versions

Only the latest release (v1.x) receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it immediately:
- **Do not** open a public issue.
- Email your findings directly to the project maintainers.
- Provide a clear description, proof of concept, and steps to reproduce.
- We will acknowledge receipt of your report within 48 hours and coordinate a public release fix.
