# Contributing to AI Code Review Platform

First off, thank you for considering contributing to the AI Code Review Platform!

## How Can I Contribute?

### Reporting Bugs
- Use GitHub Issues to report bugs.
- Include a clear description of the bug, steps to reproduce, and any relevant logs or screenshot references.

### Suggesting Enhancements
- Open a GitHub Issue explaining the desired feature, why it is useful, and potential implementation outlines.

### Pull Requests
- Fork the repository.
- Create a new feature branch (`git checkout -b feature/amazing-feature`).
- Ensure all tests pass (`pytest` in backend).
- Follow clean architecture boundaries (isolating database, routes, logic, and schemas).
- Format code according to standards (PEP-8 for Python, ESLint/Prettier for React).
- Open a Pull Request targeting `main`.
